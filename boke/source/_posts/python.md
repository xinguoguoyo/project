``` bash
一、python的特点
     1.有最庞大的免费代码库，可以给初学者足够的资源;
     2.他是人工智能，大数据分析的重要支持;
     3.他是一个脚本语言和其他编程语言相比更简洁，高效;
     
     
     
     debug:解决程序报错

               ![Demo](微信图片_20191017165151)
               
               
               
     变量和赋值:
                变量就像一个带着标签的盒子，需要你把数据放进去。
                name = '小千'
                name = '大千'
                print(name)
                终端出现的是大千，因为一个变量有最大的容量，不可无限制的塞东西，代码的世界盒子很小，容量仅为1，
                当程序运行时，首先读取的是小千，当运行到第二行时读取的是大千，print输出的数值就是最新的大千。
                
                
     变量命名的规范:
                    1.只能是一个词;
                    2.只能含有字符，数字和下划线;
                    3.不能以数字开头;
                    4.尽量描述包含的数据内容;
                    5.不要使用python函数或关键字。
                    除了变量命名之外，还需要主要的是:在代码的世界里，赋值符号  =    不是左边等于右边的意思，仅仅表示赋值
                    的动作，把右边的内容放到左边里的盒子中的意思。
                    代表左右两边相等的符号，是比较运算符 = =，虽然他俩长得像，但是代表的意思完全不同。
                    
```
